# Luxury Furniture E-Commerce Website

A modern furniture e-commerce web application built using **React, TypeScript, and Vite**.
This project is developed with a focus on clean UI, scalable structure, and backend integration.

## Features
- Home page with featured furniture collections
- Product listing and category-based filtering
- Product details page
- Cart and wishlist functionality
- User authentication pages (Login / Signup)
- Admin dashboard (UI)
- Responsive design for all screen sizes

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Context API

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally (development)

Start backend (from `backend/`):

```bash
cd backend
npm install
npm run dev
```

Start frontend (project root):

```bash
npm install
npm run dev
```

### Build (production)

```bash
npm run build
```

### Notes
- Copy `.env.example` to `.env` and provide values before running the backend.
- `build/` artifacts are ignored by default; CI will still produce builds for releases.
